Deploy to GitHub Pages

Repository:
https://github.com/Dkarch2026/project-management-decision-simulator

Upload these files and folders from this github-pages folder:
- index.html
- .nojekyll
- README.md
- src/

GitHub web upload steps:
1. Open the repository on GitHub.
2. Click "uploading an existing file" or Add file > Upload files.
3. Drag the contents of this github-pages folder into the upload area.
4. Commit the files to the main branch.
5. Go to Settings > Pages.
6. Under Build and deployment, choose "Deploy from a branch".
7. Select branch: main.
8. Select folder: / (root).
9. Click Save.

The site URL should become:
https://dkarch2026.github.io/project-management-decision-simulator/

It can take a minute or two for GitHub Pages to publish after the first setup.
